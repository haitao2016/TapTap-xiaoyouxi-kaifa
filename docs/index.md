# TapDev Studio

跨平台 TapTap 小游戏集成开发环境

## 特性

- 🎮 **Unity 游戏开发** - 深度集成 Unity 引擎，支持一键构建
- 🛠️ **强大的代码编辑器** - 基于 Monaco 的专业代码编辑体验
- 🐛 **实时调试** - 支持断点调试、变量监视、调用堆栈
- 🎨 **主题系统** - 支持亮/暗/自定义主题
- 🔌 **插件生态** - 丰富的插件系统，支持自定义扩展
- 📱 **跨平台支持** - 支持 Windows、macOS、iOS、Android

## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build
```
