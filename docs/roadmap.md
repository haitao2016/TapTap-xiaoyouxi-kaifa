## 贡献指南

### 开发环境设置

```bash
# 克隆项目
git clone https://github.com/tapdev/studio.git
cd studio

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 运行测试
pnpm test

# 类型检查
pnpm typecheck
```

### 代码规范

- 使用 TypeScript strict 模式
- 遵循 ESLint 规则
- 提交前运行 `pnpm typecheck`
- 编写单元测试

### 提交规范

```
feat: 新功能
fix: 修复问题
docs: 文档更新
style: 代码格式
refactor: 重构
test: 测试相关
chore: 构建/工具相关
```
