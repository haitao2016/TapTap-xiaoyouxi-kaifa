# 快速开始

## 创建新项目

1. 打开 TapDev Studio
2. 点击「新建项目」按钮
3. 选择项目模板（如 Unity 2D 模板）
4. 设置项目名称和保存路径
5. 点击「创建」

## 打开现有项目

1. 点击「打开项目」按钮
2. 选择项目目录
3. TapDev Studio 会自动识别项目类型

## 基本操作

### 编辑代码

- 使用左侧文件树浏览项目文件
- 点击文件在编辑器中打开
- 支持语法高亮、代码补全、多光标编辑

### 构建项目

1. 打开构建面板（快捷键：Ctrl+B）
2. 配置构建参数：
   - 目标平台（WebGL、Android、iOS）
   - 构建类型（开发/生产）
   - 输出路径
3. 点击「开始构建」

### 调试游戏

1. 点击工具栏的「启动调试」按钮
2. 设置断点（点击行号）
3. 使用调试面板查看变量和调用堆栈

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+S` | 保存文件 |
| `Ctrl+B` | 打开构建面板 |
| `F5` | 启动调试 |
| `F10` | 单步跳过 |
| `F11` | 单步进入 |
| `Ctrl+Shift+P` | 打开命令面板 |
