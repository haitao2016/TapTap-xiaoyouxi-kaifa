# 开发计划

本目录包含 TapDev Studio 的开发计划和相关文档。

## 📁 文件说明

| 文件 | 说明 |
|------|------|
| `development-plan.json` | 开发计划主文件，包含所有任务和状态 |
| `dev-plan.js` | 开发计划管理脚本 |

## 🚀 使用方法

### 查看开发计划概览

```bash
node scripts/dev-plan.js
```

### 查看详细状态

```bash
node scripts/dev-plan.js --status
```

### 查看下一个待办任务

```bash
node scripts/dev-plan.js --next
```

### 标记任务完成

```bash
node scripts/dev-plan.js --complete P1-1
```

### 保存当前状态

```bash
node scripts/dev-plan.js --save
```

## 📝 任务 ID 格式

任务 ID 格式: `P{阶段}-{序号}`

例如:
- `P1-1` = Phase 1 的第 1 个任务
- `P2-3` = Phase 2 的第 3 个任务

## 🔄 自动更新

开发计划会在以下情况下自动保存:
- 使用 `--complete` 标记任务完成时
- 使用 `--save` 命令显式保存时

建议定期运行 `--save` 保存最新状态。
