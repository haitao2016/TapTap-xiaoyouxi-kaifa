环境要求

在开始使用 TapDev Studio 之前，请确保你的开发环境满足以下要求：

  • Node.js >= 18.0.0（推荐使用 LTS 版本）
  • pnpm >= 8.0.0（包管理器）
  • Unity 2021.3+（如使用 Unity 引擎）
  • TapTap 小游戏 Unity SDK（最新稳定版）

安装步骤

按照以下步骤完成开发环境的搭建：

  1. 安装 pnpm 包管理器
  2. 克隆项目并安装依赖
  3. 启动开发服务器
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

常见问题

Q: pnpm install 时出现网络超时怎么办？

A: 可以尝试切换 npm 镜像源，推荐使用淘宝镜像：pnpm config set registry https://registry.npmmirror.com

Q: Unity SDK 版本不兼容如何处理？

A: 请确保 Unity 版本为 2021.3 或更高。如果仍然不兼容，可以在 TapTap 开发者后台下载对应版本的 SDK，或联系技术支持获取帮助。

Q: 开发服务器启动后页面空白？

A: 请检查浏览器控制台是否有报错信息。常见原因包括端口被占用（默认 3000），可尝试使用 pnpm dev --port 3001 切换端口。
